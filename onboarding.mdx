##

To get started, use the steps below:

<Steps>
  <Step title="Step 1:  Sign up for a Producta account">
    On the [home page](https://producta.ai/), click on the [**Sign Up**](https://accounts.producta.ai/sign-up?redirect_url=https%3A%2F%2Fdashboard.producta.ai%2F) button.
  </Step>

  <Step title="Step 2: Follow onboarding steps">
    Follow the onboarding steps to connect your GitHub and Linear accounts.

    <Note>
      You should only install one repository at a time.
    </Note>
  </Step>

  <Step title="Step 3: Check Linear board">
    Check your new Linear boards for two new states. It should appear as shown in the image below:

    ![](/images/producta-linear-states\(3\).png)
  </Step>
</Steps>

<Tip>
  If you mistakenly installed multiple GitHub repos, navigate to **Settings** > **Applications** > **Installed GitHub Apps**. Then, click on **Configure.** Select **Only select repositories**, and choose a repo for your use case.
</Tip>

Check out [this guide](https://chisom.mintlify.app/producta/goodtask) to learn how to use Producta.
